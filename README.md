# Iskonproject
This project is a web application developed for ISKCON to provide information about the organization, its philosophy, activities, and events. The website is built using the Flask framework and focuses on delivering structured, easy-to-access content for visitors.

This repository contains the website source (templates, static files, and a small Flask backend) ready to be deployed or further developed.
